Lettuce Disease Detection App

An Android mobile application that helps users detect lettuce diseases through image classification and provides treatment recommendations.

GitHub Repository Back to Portfolio

Project Overview

Description

The Lettuce Disease Detection App is designed to assist users in identifying possible lettuce diseases using a mobile camera or selected image. The app uses TensorFlow Lite for on-device image classification and connects to a backend API to retrieve treatment protocols and save diagnosis records.
